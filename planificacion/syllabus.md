## Descripción del Curso

Curso introductorio de Sistemas Operativos que profundiza en los fundamentos teóricos y prácticos. Los estudiantes aprenderán sobre arquitectura de SO, gestión de procesos (con programación en C de system calls), planificación, sincronización, gestión de memoria, sistemas de archivos y E/S.

---

## Unidades Temáticas

### UNIDAD 1: Introducción (6 temas)
1. Definición y funciones del SO
2. Evolución histórica
3. Arquitectura (monolítica, capas, microkernel)
4. Servicios del sistema operativo
5. Llamadas al sistema
6. Estructura de un SO

### UNIDAD 2: Gestión de Procesos (6 temas)
1. Concepto de proceso vs thread
2. Modelo de proceso (PCB, estados, transiciones)
3. Operaciones con procesos
4. Threads: modelos, niveles
5. Comunicación entre procesos (IPC)
6. Problemas clásicos de IPC

### UNIDAD 3: Planificación de Procesos (6 temas)
1. Conceptos de planificación
2. Criterios de planificación
3. Algoritmos de planificación
4. Planificación en tiempo real
5. Planificación en multiprocesadores
6. Evaluación de algoritmos

### UNIDAD 4: Sincronización y Deadlocks (6 temas)
1. Sección crítica
2. Hardware de sincronización
3. Semáforos
4. Monitores
5. Problemas clásicos
6. Deadlock: modelo, prevención, detección

### UNIDAD 5: Gestión de Memoria (13 temas)
1. Direcciones lógicas vs físicas
2. Binding de direcciones
3. Carga dinámica
4. Overlays
5. Swapping
6. Partición de memoria
7. Paginación
8. Segmentación
9. Segmentación con paginación
10. Memoria virtual
11. Demanding paging
12. Thrashing
13. Algoritmos de reemplazo

### UNIDAD 6: Gestión de Memoria Secundaria (7 temas)
1. Estructura de discos
2. Planificación de disco
3. Gestión de espacio
4. RAID
5. Optimización de disco
6. Caching de disco
7. Almacenamiento sólido

### UNIDAD 7: Sistemas de Archivos (8 temas)
1. Concepto de archivo
2. Atributos y tipos
3. Operaciones de archivo
4. Métodos de acceso
5. Estructura de directorios
6. Montaje de sistemas de archivos
7. Implementación de sistemas de archivos
8. Ejemplos (FAT, NTFS, ext4)

### UNIDAD 8: Entrada/Salida y Protección (8 temas)
1. Hardware de E/S
2. Polling vs Interrupt driven vs DMA
3. E/S en aplicaciones
4. Kernel de E/S
5. Subsistema de E/S
6. Conceptos de protección
7. Dominios de protección
8. Capabilities
