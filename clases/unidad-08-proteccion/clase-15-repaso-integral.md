## Preguntas Clave

## Ejercicio 1: Planificación

- P1 (llega 0, CPU 8)
- P2 (llega 1, CPU 4)
- P3 (llega 2, CPU 9)
- P4 (llega 3, CPU 5)

Tareas:
- FCFS, SJF, RR (q=2)
- Calcular Turnaround y Waiting Time
